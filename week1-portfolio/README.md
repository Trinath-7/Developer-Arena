# Personal Portfolio Website

A modern, fully responsive, and professional personal developer portfolio. Built to showcase projects, skills, and contact details with a minimalist aesthetic and a dynamic dark-themed design accented in red.

## Features
- **Responsive Layout**: Adopts a mobile-first philosophy ensuring seamless view across smartphones, tablets, and desktops using detailed media queries.
- **Dark Theme Environment**: Visual structure emphasizing deep black (`#000`) and high contrast red (`#f00`) highlighting per requirements.
- **Accessible & Semantic HTML5**: Leveraging headers, nav, main, sections, and footers. Includes basic alt attributes and aria labels.
- **Dynamic JavaScript Navigation**: Implements an interactive hamburger menu tailored for mobile touch spaces. Includes active screen link highlighting on scroll.
- **Client-Side Form Validation**: Complete mock layout of contact functionality containing required validation patterns.

## Tech Stack
- **HTML5:** Pure and structural semantic layout
- **CSS3:** Heavy use of modern spec paradigms: Variables, Flexbox, Grid, nested logic logic (mocked via structured scoping), transitions.
- **JavaScript (Vanilla):** DOM manipulation tailored for menu handling and scrolling visual updates. No libraries required.
